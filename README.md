# 🚢 Last Ship Out

A cinematic 3D psychological assessment game for React Native, built with Expo, Three.js, and React Three Fiber.

## 🎮 Game Overview

An immersive mobile experience that functions as a stealth psychological profiling tool for the MayaAI dating platform. Players make decisions under pressure that reveal authentic personality constructs.

## 🛠 Tech Stack

- **React Native** (New Architecture/Fabric)
- **Expo SDK 52+**
- **Three.js + React Three Fiber** - 3D rendering
- **@react-three/drei** - 3D helpers
- **@react-three/postprocessing** - Visual effects
- **Zustand** - State management
- **MMKV** - Ultra-fast storage for telemetry
- **TypeScript** - Type safety
